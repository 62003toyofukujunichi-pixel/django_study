# Django 基礎学習リポジトリ

Django の基本構造を学ぶために作成した練習用プロジェクトです。

## 学んだこと
- Django プロジェクトとアプリの作成方法
- サーバーの起動方法（manage.py runserver）
- 基本的なルーティング（urls.py）とビュー（views.py）の連携
- HTML テンプレートへのデータ受け渡し

## 備考
このリポジトリで基礎を固めた後、応用として「アドレス帳アプリ（address_book_django）」を作成しました。



## 開発環境・使用ツール
- **Python / Django**: Webフレームワーク
- **PyCharm**: IDE（統合開発環境）
- **Warp (Terminal)**: AI機能を搭載したモダンターミナルを使用し、効率的なコマンド操作とエラー解決を実施
- **Git / GitHub**: バージョン管理およびコード管理
